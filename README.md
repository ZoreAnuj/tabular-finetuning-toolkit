# Tabular Finetuning Toolkit

A streamlined library for inference and fine-tuning of tabular foundation models. This project adapts and simplifies the original TabTune codebase to provide a more accessible toolkit for experimenting with tabular data using modern foundation model approaches.

## Key Features
*   Unified interface for inference with pre-trained tabular models.
*   Tools for fine-tuning models on custom datasets.
*   Simplified data preprocessing and evaluation pipelines.

## Tech Stack
*   Python
*   PyTorch
*   Hugging Face Transformers
*   Scikit-learn

## Getting Started
```bash
git clone https://github.com/zoreanuj/tabular-finetuning-toolkit.git
cd tabular-finetuning-toolkit
pip install -r requirements.txt
```
